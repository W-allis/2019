前言

根据《中华人民共和国信托法》、《信托公司管理办法》、《信托公司集合资金信托计划管理办法》、《关于规范金融机构资产管理业务的指导意见》及其他法律法规的规定，投资者与受托人双方遵循平等、自愿和诚实信用原则，为明确各自的权利义务，在充分协商的基础上，就设立信托事宜达成一致，特订立本合同，以资遵守。

信托合同是规定信托合同当事人之间权利义务的基本法律文件，其他与本信托相关的涉及信托合同当事人之间权利义务关系的任何文件或表述，均以信托合同为准。投资者自签署认购风险申明书、信托合同，交付认购资金，并于信托计划成立之日起成为本信托的委托人/受益人之一，信托合同当事人按照《中华人民共和国信托法》、《信托公司管理办法》、《信托公司集合资金信托计划管理办法》、《关于规范金融机构资产管理业务的指导意见》及其他有关法律法规的规定和信托文件的约定享有权利、承担义务。

受托人将恪尽职守，履行诚实、信用、谨慎、有效管理的义务，但并不保证信托财产的运用无风险，也不保证受益人的最低收益。

受托人在信托合同之外披露的涉及本信托的信息，其内容涉及界定信托合同当事人之间权利义务关系的，应以信托合同为准。

 

第1条 释义

在信托文件中，除上下文另有解释，下列词语具有以下含义： 

1.1 本合同 / 信托合同： 指委托人与受托人签订的《发生发的》及对该合同的任何有效修订和补充。

1.2 说明书 / 信托计划说明书： 指《》及对该说明书的任何有效修订和补充。

1.3 认购风险申明书： 指投资者认购信托单位时签署的《》及对该申明书的任何修订和补充。

1.4 信托文件： 指信托合同、信托计划说明书、认购风险申明书的总称。

1.5 本信托 / 信托计划： 指受托人根据信托文件设立的“”。

1.6 受托人 / 云南信托： 指云南国际信托有限公司。

1.7 委托人： 指签署信托文件、按照约定交付认购资金的合格投资者。本信托计划的委托人分为优先级委托人与一般级委托人。

1.8 优先级委托人： 指认购本信托计划项下优先级信托单位并享有优先级受益权的委托人。

1.9 一般级委托人： 指认购本信托计划项下一般级信托单位并享有一般级受益权的委托人。

1.10 受益人： 指在本信托计划项下合法享有信托受益权的自然人、法人或者依法成立的其他组织。本信托计划的受益人分为优先级受益人与一般级受益人。（ 特别提示：（1）在本信托设立时，参与信托计划的委托人为唯一受益人。（2）在本信托成立后，如信托受益权发生转让，则受益人为以受让或其他合法方式取得信托受益权的自然人、法人或者依法成立的其他组织。）

1.11 优先级受益人： 指持有优先级信托单位，享有优先级受益权的人。

1.12 一般级受益人： 指持有一般级信托单位，享有一般级受益权的人。

1.13 投资者 / 合格投资者： 指符合《关于规范金融机构资产管理业务的指导意见》等法律法规规定和信托文件约定，能够识别、判断和承担信托计划相应风险的投资者。

1.14 信托受益权： 指受益人在信托计划中享有的权利，包括但不限于取得受托人分配信托利益的权利。本信托计划项下划分为优先级受益权与一般级受益权，优先级受益权由优先级受益人享有，一般级受益权由一般级受益人享有。优先级受益权对应的信托利益在分配顺序上优先于一般级受益权。

1.15 信托单位： 指信托受益权的份额化表现形式，信托计划项下的信托受益权划分为等份额的信托单位；在信托计划成立时，每份信托单位对应的认购资金为人民币壹元（RMB￥1）。

1.16 优先级信托单位： 指优先级受益权的份额化表现形式；信托计划向信托文件规定的合格投资者发行优先级信托单位。

1.17 一般级信托单位： 指一般级受益权的份额化表现形式；信托计划向信托文件规定的合格投资者发行一般级信托单位。

1.18 信托单位总份数： 指信托计划项下的信托单位份额总数。

1.19 认购： 指投资者在信托计划推介期内通过购买信托单位而参与信托计划的行为。

1.20 认购资金： 指各投资者按照信托文件的约定因认购信托单位而交付给受托人的资金。

1.21 信托资金 / 信托本金： 指根据信托文件约定，各委托人于信托计划成立时交付给受托人并进入信托财产专户的资金。

1.22 信托财产： 指信托资金及受托人对信托资金管理、运用、处分所取得的财产及损益的总和。

1.23 信托财产总值： 指信托项下的全部信托财产价值之和。

1.24 信托财产净值： 指信托财产总值扣除应付未付信托费用和其余负债后的余额。

1.25 信托单位净值： 指信托财产净值除以存续信托单位。

1.26 信托费用： 指按本合同第条约定、由信托财产承担的费用。

1.27 信托利益： 指受益人因享有信托受益权而取得的受托人分配的信托财产，包括信托本金和信托收益。

1.28 核算日： 指受托人根据信托合同的约定在本信托计划存续期限内和本信托计划终止时核算信托费用及向受益人核算信托利益之日。具体指：（1）本信托项下的首个核算日为【】；（2）最后一个核算日即为本信托终止日；（3）其余核算日为【】。受托人有权根据实际情况调整核算日的时间。

1.29 分配日： 指每个核算日后【】个工作日（含本数）内的任意一日。

1.30 信托计划成立日： 指信托计划成立的当日，即在信托文件约定的信托计划成立条件全部满足后，受托人宣告的信托计划成立日期。

1.31 信托计划终止日： 指信托计划期限届满终止、提前终止或延期终止之日。

1.32 盯控日： 指信托成立后的每个 ，该日非工作日的顺延至下一个。

1.33 推介期： 指受托人就信托计划向合格投资者进行推介的期间。受托人有权根据具体情况延长或提前终止信托计划推介期。

1.34 预定信托期限： 指信托计划预定的存续期限，具体指本合同第条约定的存续期限。

1.35 信托计划存续期限 / 存续期限： 指信托计划成立日至信托计划终止日的时间段，具体指本合同第条约定的存续期限。

1.36 当期： 指信托存续期间内，就信托利益、信托管理费及其他相关信托费用的核算而言，为确定相邻核算日之间以及核算日与邻近的信托成立日、信托终止日之间的期间而设定的概念，计算当期的天数时应当包含期间的起始日但不包含期间的终止日。如，就信托利益核算而言，当期为：前一个核算日（含）至本核算日（不含）的期间；但第一个当期为信托成立日（含）至第一个核算日（不含）的期间；最后一个当期为信托终止日（不含）至该日前最近一个核算日（含）的期间。

1.37 循环期： 指受托人将还款资金【】向借款人持续发放贷款的期间，具体指本信托计划成立日起至的期间。

1.38 摊还期： 指受托人不再发放新的贷款，将全部回款资金用于支付信托费用及分配信托利益的期间。具体包括（1）循环期结束后至信托终止的期间；（2）因发生风险预警事件而提前结束循环期之日起（含该日）至信托终止的期间。

1.39 信托财产专户： 指受托人在保管人处为信托计划开立的专用银行账户，也简称信托专户。

1.40 信托利益收付账户： 指受益人指定的用于接收受托人分配的信托利益的银行账户。

1.0 收益账户： 系指本信托计划项下与本金账户并列的虚拟核算账户，本信托计划将收到的借款利息计入该账户。

1.0 本金账户： 系指本信托计划项下与收益账户并列的虚拟核算账户，本信托计划将收到的借款本金计入该账户。

1.41 借款人 / 债务人： 指本信托项下与受托人签订《贷款合同》的自然人借款人的简称。

1.42 《信托贷款合同》 / 《贷款合同》： 指受托人根据信托文件的约定，就信托资金的管理、运用而与借款人签署的编号为的《》及对该合同的任何有效修订和补充。

1.43 贷款： 指受托人根据《贷款合同》的约定向借款人发放的个人贷款。

1.44 还款日： 指借款人根据《贷款合同》的约定偿还贷款的日期，本信托项下每一笔贷款可有多个还款日。

1.45 资产服务顾问： 指本信托项下对信托贷款提供资产服务的机构。本信托设立时的资产服务顾问为公司。

1.46 《资产顾问服务协议》： 指受托人与资产服务顾问签署的编号为【】的《》及其附件、对该协议的任何有效修订或补充。

1.47 信息技术服务机构： 指本信托项下对信托贷款提供信息管理服务的机构，即杭州随地付网络技术有限公司。

1.48 《信息技术服务协议》： 指受托人与信息技术服务机构签署的编号为【】的《》及其附件、对该协议的任何有效修订或补充。

1.49 保管人： 指。

1.50 《保管合同》： 指受托人与保管人签署的编号为【】的《》及其任何有效修改和补充。

1.51 交易文件： 指《贷款合同》、《资产顾问服务协议》、《信息技术服务协议》、《保管合同》所列举文件的统称。

1.52 风险预警事件： 系指本合同第8.3条第（1）款约定的情况。

1.53 丧失清偿能力事件： 系指本合同第8.4条约定的情况。【根据项目具体情况进行添加】

1.56 法律法规： 指中国（仅就本合同而言，不包括香港、澳门、台湾地区）现行有效并公布实施的法律、行政法规、部门规章、司法解释及监管部门的决定、通知等。

1.57 工作日： 指中华人民共和国国务院规定的金融机构正常营业日，不含中国的法定公休日和节假日。

第2条 信托目的

委托人基于对受托人的信任，自愿以其合法所有的资金认购信托单位并加入信托计划，同意由受托人在本合同确定的范围内按照信托文件的约定管理、运用、处分信托财产，以实现信托财产的保值和增值，并按信托文件约定向受益人支付信托利益。

第3条 受益人

3.1 信托计划成立时是自益信托，受益人与委托人为同一人。

3.2 本信托计划项下信托受益权区分为优先级受益权与一般级受益权两类。受益人根据本信托合同的约定享有优先级受益权或一般级受益权。优先级受益权与一般级受益权并非受托人做出的对该权利可能获取的全部或部分信托利益分配的承诺、保证、保障、担保。
